# Fallback Voice Patterns

These are sensible defaults for the ghost-writer skill when no user-level voice guide has been created yet. They represent a neutral, competent technical-writing voice that avoids the most common AI-writing tells. Replace them as soon as you have real samples from the user.

**These are temporary.** Tell the user you're using defaults and that running Voice Discovery will produce much better results.

## Default register

Use a single narrative register: sequential, concrete, first-person. Walk through events in order. Describe what happened, not how it felt.

## Default sentence rules

- Lead with the action, not the observation about the action.
- Avoid starting sentences with "But," "However," or "It's worth noting."
- Don't use "not X, but Y" framing — state the positive claim first.
- Don't over-hedge. If the user said something confidently, draft it confidently.

## Default anti-patterns

- No staccato fragments for dramatic effect ("Did the thing. Loved it. Moved on.")
- No extended metaphors across paragraphs.
- No "As a [role], I..." openers.
- No "Let me explain..." or "Here's the thing..." preambles.
- No emoji.

## Default formatting

- Markdown with reference-style links.
- Footnotes for citations and technical terms.
- First-person throughout.

## Default reading level

Target 10th–12th grade Flesch-Kincaid. Use proper terminology but explain it once, plainly. Don't write academically.

## Default length

Lean shorter. If unsure whether to expand or cut, cut.
