# Opener Rubric

Openers set the sentence's tone and direction. Poor openers bury the point or signal uncertainty when none is warranted. These sections flag the patterns David's voice rejects: hedges that undercut confidence, the "but it was" construction, "not X but Y" framing, and filler-first openers that waste the first word.

## Hedge-first openers

Sentences that lead with doubt or tentative language, signaling uncertainty when you should plant the flag first:

- I think
- Maybe
- Perhaps
- It seems
- It might be worth noting
- I'd argue
- In my view
- One could say
- It's possible that
- I suspect

These soften the claim before stating it. David leads with the verb or the action, not the feeling. Answers have weight.

## "But it was" openers

The construction: "But it was..." — leading with negation before the actual point. From David's voice guide: reject this construction.

- But it was the best fit
- But it was a good point
- But it was wrong
- But it was worth the cost
- But it was actually elegant
- But it wasn't what we expected

This pattern contradicts what came before. Plant the flag first, then address the contradiction. "It was the best fit — but expensive" lands harder than "But it was the best fit — too pricey."

## "Not X but Y" framing

Sentences that lead with what something isn't before saying what it is. The rule: name it properly first, then analogize or contrast.

- Not a tutorial, but a guide
- Not about speed, but about precision
- Not technical debt, but evolution
- Not a feature, but a philosophy
- Not criticism, but observation
- Not a framework, but a pattern

State the point. Address counters after, if at all. Don't preface with negation to establish false confidence.

## Filler-first openers

Sentences that start with discourse markers — words that add no information and waste the slot:

- So,
- Well,
- Look,
- Honestly,
- Basically,
- Essentially,
- Actually,
- Clearly,

These are conversational tics. The sentence works fine without them. The first word should carry weight.
